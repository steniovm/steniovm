1 Crie uma página html para uma calculadora simples utilizando formulários e funções. Estruture o seu trabalho como um projeto (arquivos separados para html, css e javascript). Sua calculadora deve seguir as seguintes regras:

    ⦁ Um campo para receber o primeiro operador.

    ⦁ Um campo para receber o segundo operador.

    ⦁ Um botão para cada operação: Soma, Subtração, Divisão e Multiplicação.

    ⦁ O resultado deve ser exibido em uma tag de texto.


2 Publique o material do exercício 1 no seu github.
